# Get Schema

Retrieve details of a specific schema by its ID.

## API Request

```bash
GET https://withflora.io/v1/schemas/{id}
```

## Headers

<ParamField header="Authorization" required>
  Bearer YOUR_FLORA_API_KEY
</ParamField>

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the schema to retrieve
</ParamField>

## Response

<ResponseField name="status" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="message" type="string">
  A message describing the result
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="id" type="string">
      Unique identifier for the schema
    </ResponseField>
    <ResponseField name="name" type="string">
      Name of the schema
    </ResponseField>
    <ResponseField name="description" type="string">
      Optional description of the schema
    </ResponseField>
    <ResponseField name="createdAt" type="string">
      Timestamp when the schema was created
    </ResponseField>
    <ResponseField name="structure" type="object">
      JSON Schema structure defining the data format
    </ResponseField>
  </Expandable>
</ResponseField>

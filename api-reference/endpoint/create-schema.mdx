# Create Schema

Create a new schema to define the structure of your data.

## API Request

```bash
POST https://withflora.io/v1/schemas
```

## Headers

<ParamField header="Authorization" required>
  Bearer YOUR_FLORA_API_KEY
</ParamField>

<ParamField header="Content-Type" required>
  application/json
</ParamField>

## Body Parameters

<ParamField body="name" type="string" required>
  Name of the schema
</ParamField>

<ParamField body="structure" type="object" required>
  JSON Schema structure defining your data format. This should follow JSON Schema standards and include:
  - `type`: The data type (object, array, etc.)
  - `properties`: Object containing field definitions
  - `required`: Boolean indicating if the schema is required
</ParamField>

## Response

<ResponseField name="status" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="message" type="string">
  A message describing the result
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="id" type="string">
      Unique identifier for the created schema
    </ResponseField>
    <ResponseField name="name" type="string">
      Name of the schema
    </ResponseField>
    <ResponseField name="description" type="string">
      Optional description of the schema
    </ResponseField>
    <ResponseField name="createdAt" type="string">
      Timestamp when the schema was created
    </ResponseField>
    <ResponseField name="structure" type="object">
      JSON Schema structure defining the data format
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash
curl --request POST \
  --url 'https://withflora.io/v1/schemas' \
  --header 'Authorization: Bearer YOUR_FLORA_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "name": "News Articles",
    "structure": {
      "type": "array",
      "required": true,
      "items": {
        "type": "object",
        "properties": {
          "title": {
            "type": "string",
            "describe": "The article title"
          },
          "summary": {
            "type": "string",
            "describe": "The news summary"
          },
          "link": {
            "type": "string",
            "describe": "URL to learn more"
          }
        }
      }
    }
  }'
```
</RequestExample>

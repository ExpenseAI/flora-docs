# List Schemas

Get a paginated list of all schemas in your account.

## API Request

```bash
GET https://withflora.io/v1/schemas
```

## Headers

<ParamField header="Authorization" required>
  Bearer YOUR_FLORA_API_KEY
</ParamField>

## Query Parameters

<ParamField query="page" type="integer">
  Page number for pagination
</ParamField>

<ParamField query="perPage" type="integer">
  Number of items per page
</ParamField>

## Response

<ResponseField name="status" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="message" type="string">
  A message describing the result
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="data" type="array">
      Array of schema objects
      <Expandable title="Schema Object">
        <ResponseField name="id" type="string">
          Unique identifier for the schema
        </ResponseField>
        <ResponseField name="name" type="string">
          Name of the schema
        </ResponseField>
        <ResponseField name="description" type="string">
          Optional description of the schema
        </ResponseField>
        <ResponseField name="createdAt" type="string">
          Timestamp when the schema was created
        </ResponseField>
        <ResponseField name="structure" type="object">
          JSON Schema structure defining the data format
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="meta" type="object">
      <Expandable title="properties">
        <ResponseField name="total" type="integer">Total number of items</ResponseField>
        <ResponseField name="perPage" type="integer">Items per page</ResponseField>
        <ResponseField name="currentPage" type="integer">Current page number</ResponseField>
        <ResponseField name="lastPage" type="integer">Last page number</ResponseField>
        <ResponseField name="firstPage" type="integer">First page number</ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash
curl --request GET \
  --url 'https://withflora.io/v1/schemas?page=1&perPage=10' \
  --header 'Authorization: Bearer YOUR_FLORA_API_KEY'
```
</RequestExample>

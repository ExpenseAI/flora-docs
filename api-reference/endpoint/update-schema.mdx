# Update Schema

Update an existing schema's details and structure.

## API Request

```bash
PUT https://withflora.io/v1/schemas/{id}
```

## Headers

<ParamField header="Authorization" required>
  Bearer YOUR_FLORA_API_KEY
</ParamField>

<ParamField header="Content-Type" required>
  application/json
</ParamField>

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier of the schema to update
</ParamField>

## Body Parameters

<ParamField body="name" type="string">
  New name for the schema
</ParamField>

<ParamField body="structure" type="object">
  Updated JSON Schema structure defining your data format. This should follow JSON Schema standards and include:
  - `type`: The data type (object, array, etc.)
  - `properties`: Object containing field definitions
  - `required`: Boolean indicating if the schema is required
</ParamField>

## Response

<ResponseField name="status" type="boolean">
  Indicates if the request was successful
</ResponseField>

<ResponseField name="message" type="string">
  A message describing the result
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="properties">
    <ResponseField name="id" type="string">
      Unique identifier for the schema
    </ResponseField>
    <ResponseField name="name" type="string">
      Updated name of the schema
    </ResponseField>
    <ResponseField name="description" type="string">
      Optional description of the schema
    </ResponseField>
    <ResponseField name="createdAt" type="string">
      Timestamp when the schema was created
    </ResponseField>
    <ResponseField name="structure" type="object">
      Updated JSON Schema structure
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash
curl --request PUT \
  --url 'https://withflora.io/v1/schemas/c622793c-5e69-4670-bfa5-6e57b6fcbc74' \
  --header 'Authorization: Bearer YOUR_FLORA_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "name": "Updated Product Object",
    "structure": {
      "type": "object",
      "required": true,
      "properties": {
        "amount": {
          "type": "number",
          "required": true,
          "describe": "Total amount of the purchased items"
        },
        "currency": {
          "type": "string",
          "required": true,
          "minLength": "3",
          "describe": "The ISO code of the currency e.g. NGN, USD"
        },
        "storeName": {
          "type": "string",
          "required": true,
          "describe": "Name of the store"
        },
        "date": {
          "type": "string",
          "format": "date-time",
          "describe": "Purchase date and time"
        }
      }
    }
  }'
```
</RequestExample>
